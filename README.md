# 🎵 Sistema de Gestión de Música Tipo Spotify (Modo DJ)

## 📌 Descripción

Este proyecto es una plataforma web inspirada en Spotify que permite a los usuarios buscar y seleccionar canciones, mientras que un administrador controla la reproducción en tiempo real.

El sistema está diseñado principalmente para eventos, reuniones o espacios donde múltiples usuarios interactúan con la música, pero existe un control centralizado.

---

## 🎯 Objetivo

Desarrollar una aplicación web interactiva que permita:

* La participación de usuarios en la selección de música
* El control administrativo de la reproducción
* La gestión de canciones en tiempo real
* La implementación de distintos modos de operación

---

## ⚙️ Funcionalidades principales

### 👤 Usuario público

* 🔍 Búsqueda de canciones
* ➕ Agregar canciones a la cola
* 📄 Visualización de la lista de reproducción

### 🎛️ Administrador

* ✅ Activar o desactivar canciones
* ❌ Eliminar canciones
* 🎚️ Controlar la reproducción
* 🔄 Cambiar entre modos del sistema

---

## 🎧 Modos de funcionamiento

### 🔹 Modo Normal

Las canciones se reproducen automáticamente en el orden en que fueron agregadas por los usuarios.

### 🔹 Modo DJ

El administrador tiene control total sobre qué canciones se reproducen y en qué momento.

---

## 📍 Geolocalización

El sistema utiliza geolocalización para:

* Identificar la ubicación del usuario
* Limitar el acceso a ciertas funciones según la ubicación
* Conectar usuarios dentro de una misma sesión o evento

---

## 🔄 Flujo del sistema

1. El usuario accede al sistema
2. Busca una canción
3. La agrega a la lista de reproducción
4. La información se almacena en la base de datos
5. El administrador gestiona la reproducción
6. Los cambios se reflejan en tiempo real

---

## 🧩 Tecnologías utilizadas

* 💻 Frontend: HTML, CSS, JavaScript, Bootstrap
* 🧠 Backend: PHP
* 🗄️ Base de datos: MySQL
* 🔄 Comunicación: AJAX / Fetch
* 🌐 Control de versiones: Git

✔️ Funcional

---

## ⚠️ Limitaciones

* No es un sistema de streaming como Spotify
* No incluye recomendaciones inteligentes
* Depende de fuentes externas para búsqueda de canciones

---

## 🧠 Conclusión

Este sistema cumple con el objetivo de crear una plataforma interactiva de gestión musical, enfocada en la participación de usuarios y el control administrativo en tiempo real.

No busca reemplazar Spotify, sino ofrecer una solución adaptada a eventos donde se requiere control dinámico de la música.
